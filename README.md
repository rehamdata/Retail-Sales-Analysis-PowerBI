# Retail Sales Dashboard

### 📊 Retail Sales Dashboard – Monthly Analysis (Power BI) 📌 Project Overview

This project presents an interactive monthly sales dashboard built using **Power BI**, designed to analyze sales performance and support data-driven decision-making. The dashboard provides a clear overview of key business metrics, sales trends, and customer insights through a clean and user-friendly layout.

---

### 🎯 Objectives

* **Monitor overall sales performance** on a monthly basis.
* **Analyze revenue and quantities sold** across different periods.
* **Compare sales across categories** and demographic segments, specifically focusing on **age groups**.
* **Identify top-performing trends** to gain better operational insights.
* **Ensure data integrity** by handling missing values and resolving technical conversion errors during the **ETL process**.

---

### 🛠️ Technical Highlights

* **Dynamic Sourcing**: Implemented **Parameters** (`SourcePath`) to allow easy switching of data sources between local and production environments.
* **Data Imputation**: Successfully managed a **2% missing data** rate in the age demographic column by applying mean-based imputation to maintain analysis accuracy.
* **Advanced Visuals**: Optimized UI/UX using **Tile-style Slicers** for category selection and **shadow-effect KPI cards** for better visual hierarchy.
